# 微博爬虫可视化管理系统

## 项目简介

这是一个基于Web的微博爬虫可视化管理系统，提供直观的用户界面来配置、执行和管理微博爬虫任务。系统支持多种爬取模式，提供实时任务监控、结果管理和数据可视化功能。

## 主要功能

- 📊 **仪表盘** - 系统概览、数据统计和趋势分析
- ⚙️ **配置管理** - 可视化配置爬虫参数、用户ID和输出格式
- ▶️ **任务执行** - 快速执行、定时执行和批量执行任务
- 📁 **结果管理** - 查看和管理爬取的微博数据和媒体文件
- 📝 **日志查看** - 实时查看系统运行日志
- 🔌 **API服务** - 提供RESTful API接口

## 技术栈

- **前端**: HTML5, Tailwind CSS, JavaScript, Font Awesome, Chart.js
- **后端**: Flask
- **响应式设计**: 支持桌面、平板和移动设备
- **主题切换**: 支持浅色/深色主题

## 安装和启动

### 安装依赖

```bash
# 安装Flask依赖
pip install flask werkzeug
```

### 启动服务

```bash
# 使用启动脚本启动服务
python start_server.py

# 或者直接运行service.py
python service.py

# 自定义端口启动
python start_server.py --port 8080

# 启用调试模式
python start_server.py --debug
```

服务启动后，可通过浏览器访问：`http://localhost:5000`

## 使用指南

### 配置管理

1. 在左侧导航栏点击"配置管理"
2. 输入用户ID列表（每行一个ID）
3. 选择爬取模式（全部微博或仅原创）
4. 设置爬取范围（最近N天或指定日期）
5. 选择输出格式（CSV、JSON、数据库等）
6. 配置媒体下载选项
7. 点击"保存配置"按钮

### 执行任务

1. 在左侧导航栏点击"任务执行"
2. 点击"立即执行"按钮快速开始爬取
3. 或配置定时执行计划
4. 在任务状态区域查看执行进度

### 查看结果

1. 在左侧导航栏点击"结果管理"
2. 浏览爬取的微博数据和媒体文件
3. 可下载需要的数据文件

## 深色主题

点击页面右上角的月亮图标可切换至深色主题，太阳图标可切换回浅色主题。主题设置会保存在本地，下次访问时自动应用。

## 响应式设计

系统采用响应式设计，在不同尺寸的设备上都能提供良好的用户体验：

- **桌面端**: 左侧固定导航栏，右侧内容区
- **平板端**: 自适应布局，优化显示
- **移动端**: 汉堡菜单切换导航，单列布局

## 注意事项

1. 大批量爬取可能需要较长时间，请耐心等待
2. 确保正确配置用户ID，避免爬取失败
3. 下载大量媒体文件可能会占用较多磁盘空间
4. 如需获取更多数据，可考虑配置Cookie（在数据库配置中）

## 开发说明

前端页面主要文件：
- `index.html` - 主页面文件

后端服务文件：
- `service.py` - Flask应用主文件

## 故障排除

### 服务无法启动

- 检查端口是否被占用
- 确认Python和依赖已正确安装
- 查看错误日志获取详细信息

### 爬取失败

- 检查用户ID是否正确
- 确认网络连接正常
- 查看日志获取详细错误信息

### 页面无法访问

- 确认服务已成功启动
- 检查防火墙设置
- 确认访问地址正确
